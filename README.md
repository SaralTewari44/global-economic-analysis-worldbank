global-economic-analysis-worldbank

This project analyzes sales performance data to identify trends, patterns, and key business drivers that impact revenue and performance.

## Objectives
- Analyze sales trends across time and categories
- Evaluate key performance metrics
- Identify patterns that impact business outcomes
- Support data-driven decision-making

## Tools & Methods
- Python
- pandas, numpy
- Data cleaning and transformation
- Exploratory data analysis
- Visualization

## Files
- `sales-performance-analysis.ipynb` – Main analysis notebook
